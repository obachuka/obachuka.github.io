Fat Dot

Fat Dot is a game where players try to chain explosions to reach a high score.

I made everything myself, from my own collision detection to implementing game rules.

Project Highlights:

  • Simple, single-button gameplay
  • Depth comes from the 6 different enemy types
    • Some enemy types will interact with each other
    • Avoiding one type of enemy might push players towards another enemy
    • Designed to encourage different types of movement
  • Implemented debug abilities to help with design
    • Hot-reload newly compiled code to reflect updates without restarting
    • Record and replay gameplay in real-time
    • Hot-reload during replays to see changes in action
    • Resume control at any point during replay

Specs:

  • Language
    • C++
  • Libraries
    • SDL 2.0
    • OpenGL
  • Platforms
    • Android
    • iOS
    • Windows
    • Linux
    • Chrome/Firefox

DekoBoko

DekoBoko is a fan game paying tribute the movie, My Neighbor Totoro. Gameplay is split into two parts. In the first part, players tend to their own garden. In the second part, players control the small Totoro to grow a giant tree, recreating an iconic scene from the film.

I developed the game solo.

Project Highlights:

  • Hand-drawn art and animations
  • 17 plants
  • 19 animals
  • 7 items and upgrades
  • In-game journal with entries on each plant and animal
  • Self-arranged piano cover of the movie's score

Specs:

  • Language
    • C#
  • Game Engine
    • Unity
  • Platforms
    • Android
    • iOS

Shepherd and Dog

Shepherd and Dog is a game made in 3 days for the Ludum Dare Game Jam. Players control a shepherd-and-dog team to guide their sheep back home.

I did the programming and game design, and teamed up with an artist to complete the art assets.

Project Highlights:

  • Implemented basic level editor
    • Reads ASCII text to lay out tiles in the level
    • Completed 11 levels before the deadline
  • Seemingly complex wolf behavior
    • Governed by a set of rules based off Pac-Man
    • Appears to surround the player
    • Occasionally backs off to give player breathing space

Specs:

  • Language
    • C++
  • Libraries
    • SDL 2.0
    • Box2D
  • Platforms
    • Windows
    • Chrome/Firefox

Asteroids But With Gravity

Asteroids But With Gravity is an Asteroids clone with a gravity gimmick. Players must shoot asteroids in the classic arcade style, while escaping the gravity of nearby planets.

I developed the game solo.

Project Highlights:

  • Larger playing field than original Asteroid's screen-sized area
  • Entire environment still loops around, including gravity

Specs:

  • Language
    • C++
    • Blueprints
  • Game Engine
    • Unreal Engine 4
  • Platforms
    • Windows
