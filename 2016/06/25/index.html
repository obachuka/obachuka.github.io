Practicing brush control

by Obachuka

Yesterday, I had complained about how my brush strokes looked like they were overlapping while I was trying to follow David Revoy's Krita tutorial. Today I followed Ctrl+Paint's tutorial on brush control.

The goal of this exercise is to copy the left image using nothing but brush strokes and layers. The ones on the right are mine.

It turns out, the secret to smooth looking brush strokes isn't with blending modes or pen sensitivity. It's really about abusing the eraser tool. While I did some erasing with pencil, I didn't rely on it as a primary tool, but more as a way to fix mistakes. And with ink or watercolor, I didn't have the option to erase at all. It never occurred to me to use the eraser as the actual drawing tool. I'd plant down large swatches of color using the brush, then use the eraser to create the edges I wanted, ocassionally using layers to make the process easier.

There's also a colored version of this same exercise size sheet I'll work on later. Next though, I want to improve at making clean lines. It's hard to get my lines go where I want using the tablet, and my hands tend to wobble, which is not something that happens when I use real pencils.
